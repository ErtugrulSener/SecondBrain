# Regel Nr. 1) Wenn ein sukuniertes Mim, auf ein Mim mit einer Harakat trifft, dann spricht man das Mim insgesamt 2 Harakat lang (man zieht es lang wie ein [[madd-mutasil]]).

<span style="font-size: 22pt">هُمْ مَا</span>
gesprochen "hummaa"

# Regel Nr. 2) Die erste Regel gilt auch wenn ein sukuniertes Nun auf ein Nun mit Harakat trifft.

<span style="font-size: 22pt">هُنْ نَا</span>
gesprochen "hunnnnaa"

# Regel Nr. 3) Trifft ein sukuniertes Mim auf ein Ba, so spricht man das Mim auch 2 Harakat lang.

<span style="font-size: 22pt">هُم بَا</span>
gesprochen "hummbaa"

# Regel Nr. 4) Kommt nach einem Nun mit Sukun ein Ba, so spricht man ein Mim stattdessen und hält es etwas.

<span style="font-size: 22pt">هُنْ بَا</span>
gesprochen "hummbaa"
